
# Knowledge Bomber

### List of Wikipedia Languages

Ranking is based on numbers of [qualified Wikidata items](#Qualified-Wikidata-Items) per language. 

`Coverage %` is based on Q40_Items to English percentage; `Solidness` is coverage : article ratio to simple_english as 100% benchmark; `№` is based on Q30_Items ranks; the table is sorted by Q20_Items.
